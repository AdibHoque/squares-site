ELECTRON

Electrify & Boost Up Your Server with this all-in-one bot which features Utility, Moderation, Logs, Giveaways, Games & Much more!

Command Description Usage
avatar Enlarge user Avatar! /avatar [@user]

color Get visual color image from an HEX code. /color <HEX code>
define Get word definition from Urban Dictionary /define <word>
math Evaluate mathematical inputs. /math <input>
ping Check bot ping latency! /ping
roleinfo Get information about a role. /roleinfo <role>
serverinfo Get information about the server. /serverinfo
userinfo Get information about a user. /userinfo <user>
winratecalc Calculate how many wins you need for reaching a certain Winrate. /winratecalc <current winrate> <total matches> <desired winrate>

MODERATION COMMANDS
Command Description Required Permission Usage
ban Ban a member from the server BAN_MEMBERS /ban <member ID or @member>
clearlogs Mass delete a members logs, oldest to newest sequence. MODERATE_MEMBERS /clearlogs <member> [Amount]
config Assign or change channels for certain tasks. MANAGE_CHANNELS /config <option> <value>
kick Kick a member from the server KICK_MEMBERS /kick <member ID or @member>
loginfo Log information with the attachment evidence, if any. MODERATE_MEMBERS /loginfo <Log ID>
logs See Moderation logs of a member. MODERATE_MEMBERS /logs <@member or member ID> [Filter]
mute Mute a member. MODERATE_MEMBERS /mute <member ID or @member>
nickname Change a members nickname. MANAGE_NICKNAMES /nickname <member ID or @member> <nickname>
purge Bulk delete messages of a channel. MANAGE_MESSAGES /purge <amount>
resetnick Reset a members nickname. MANAGE_NICKNAMES /resetnick <member ID or @member>
role Toggle add or remove role from a Member. MANAGE_ROLES /role <member ID or @member> <role>
unban Unban a member. BAN_MEMBERS /unban <member ID>
unmute Unmute a previously muted member. MODERATE_MEMBERS /unmute <member ID or @member>
warn Warn a member for rules violation. MODERATE_MEMBERS /warn <member ID or @member> <Reason>
embed Create a custom embed MANAGE_MESSAGES /embed <title> [author] [description] [thumbnail] [image URL]
